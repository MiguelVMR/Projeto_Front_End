Introdção ao Front-End.

A programação front-end é responsável pela construção da interface com a qual o usuário interage em um site, aplicativo ou software. Essa parte do desenvolvimento é interpretada pelos navegadores ou sistemas operacionais, e seu objetivo é transmitir os elementos visuais e interativos que permitem a interação do usuário com a aplicação. Usando de bases 3 tipos de linguagens: O JavaScript, HTML e o CSS.

O HTML

Sigla para HyperText Markup Language — Linguagem de Marcação de Hipertexto —, o HTML é o componente base da web. Isso quer dizer que ele permite a construção de websites e a inserção de novos conteúdos, como imagens e vídeos, por meio dos hipertextos.

No HTML, utilizamos elementos para definir os diferentes tipos de conteúdo de uma página web. Por exemplo, para definir um título, usamos o elemento h1 e para criar um parágrafo, usamos o elemento p. Além disso, podemos adicionar ênfase ou importância a palavras utilizando os elementos strong e em, criar listas com os elementos ul e ol, e muito mais

Imagine que você está construindo uma casa. Antes de começar a construir as paredes e colocar os móveis, é preciso ter um projeto que mostre onde cada coisa deve ficar. O HTML funciona de maneira semelhante. Ele define a estrutura de uma página web, indicando onde cada elemento deve ser colocado e como deve ser exibido

O CSS

CSS é a sigla para o termo em inglês Cascading Style Sheets que, traduzido para o português, significa Folha de Estilo em Cascatas. O CSS é fácil de aprender e entender e é facilmente utilizado com as linguagens de marcação HTML ou XHTML. O CSS ajuda a editar, alinhar, remover e trabalhar no espaço entre elementos de uma página.

O CSS é fácil de aprender e entender, e é facilmente utilizado com HTML ou XHTML. Uma das coisas que você notará sobre escrever CSS é que muito disso é sobre caixas, indicando o tamanho, cor e posição dos elementos. Isso ocorre porque muitos dos elementos HTML podem ser pensados como caixas, uma em cima das outras, e o layout CSS é baseado principalmente no modelo de caixas.

Em resumo, o CSS é uma ferramenta importante para a aparência visual do seu website, permitindo que você personalize elementos como cores, fontes e layouts. Ele pode ser facilmente aprendido e utilizado em conjunto com HTML.

Curiosidades sobre o CSS

Aqui listamos 3 curiosidades sobre o CSS

Curiosidade 1

Com o CSS você não precisa mais escrever repetidamente como os elementos individuais se parecem. Isso economiza tempo, encurta o código e diminui a chance de erros. O CSS permite que você tenha vários estilos em uma página HTML, tornando as possibilidades de personalização quase infinitas.

Curiosidade 2

Uma curiosidade interessante sobre o CSS é que muitas vezes podemos criar animações e transições apenas com o uso desta linguagem de estilo. Um exemplo bastante popular é o efeito de deslizar para cima e para baixo, que pode ser facilmente criado com CSS, sem a necessidade de bibliotecas de JavaScript ou outras ferramentas externas

Curiosidade 3

O CSS permite a criação de gradientes de cor em elementos, o que pode resultar em um efeito visual bastante interessante. Além disso, é possível utilizar propriedades como opacidade e transparência para tornar os elementos mais suaves ou mais vibrantes. Vale lembrar que, embora a linguagem seja principalmente voltada para a apresentação visual de um site, ela também pode ser utilizada para controlar o layout e a organização do conteúdo em uma página.

O JavaScript

JavaScript é uma linguagem de programação poderosa e versátil que permite adicionar interatividade e dinamismo a um site ou aplicativo web. Com algumas linhas de código, é possível controlar elementos multimídia, criar animações, validar formulários, realizar operações matemáticas e muito mais. É uma linguagem orientada a objetos e multiplataforma, o que significa que pode ser usada em diferentes sistemas operacionais e dispositivos. Além disso, o JavaScript é leve e pode ser facilmente integrado a outras tecnologias, como HTML e CSS. Com mais experiência, é possível criar projetos mais complexos, como jogos, aplicativos e até mesmo programas baseados em banco de dados.

Curiosidades sobre o JavaScript

Aqui listamos 3 curiosidades sobre o JavaScript.

Curiosidade 1

Uma curiosidade do JavaScript é que, diferentemente de outras linguagens de programação como o CoffeeScript e o TypeScript, ele é uma linguagem de script orientada a objetos multiplataforma, pequena e leve, e pode ser executada diretamente pelos navegadores web. Além disso, o JavaScript possui a capacidade de se ligar aos objetos do ambiente em que é executado, fornecendo assim um controle programático sobre eles

Curiosidade 2

Uma curiosidade do JavaScript que as outras linguagens não têm é que ele não tem um mecanismo para criar variáveis de instância encapsuladas dentro de um objeto, o que pode causar problemas ao construir programas de médio a grande porte.Além disso, TypeScript é uma linguagem que adiciona alguns elementos novos ao JavaScript e propõe melhor organização do código existente, mas todo programa JavaScript é um programa TypeScript v

Curiosidade 3

Por fim, vale lembrar que o JavaScript é uma linguagem de programação interpretada, dialeto do padrão ECMAScript, definido como Orientado a Objeto, baseado em protótipo, imperativo, fracamente tipado e dinâmico.
